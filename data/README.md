# Data Acquisition

## TCGA STAD Clinical Data

The clinical data used in this analysis can be obtained from cBioPortal:

1. Navigate to https://www.cbioportal.org/
2. Search for "Stomach Adenocarcinoma (TCGA, PanCancer Atlas)"
3. Select the study and navigate to "Download" â†’ "Clinical Data"
4. Download the clinical data TSV file
5. Rename to `tcga_2018_clinical_data.tsv` and place in this directory

Alternatively, use the GDC Data Portal: https://portal.gdc.cancer.gov/
